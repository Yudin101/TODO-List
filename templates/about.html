About

This is a TODO List app that allows you to create, edit and delete tasks. The main objective of this app is to help you organize things and make you more productive. This app offers a really simple and easy-to-use interface that can be used by anyone.

This app was initially created by me as the final project for CS50 but now I have added some new features and have made a lot of new changes. Any contribution is warmly welcomed.
